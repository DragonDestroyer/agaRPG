How this works

This entire page is static files. GitHub Pages serves the HTML/CSS/JS. When opened, PyScript downloads a full CPython interpreter compiled to WebAssembly (Pyodide) and runs real Python + Matplotlib entirely in your browser.

<script type="module" src="https://pyscript.net/releases/2025.10.2/core.js"></script>
<script type="py" config='{"packages":["matplotlib","numpy"]}' src="main.py"></script>
